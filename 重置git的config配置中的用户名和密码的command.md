1. 设置用户名 (要注意user.name是key，key空格后的是需要修改的value值)
```java
  git config --global user.name "yourname"
```

2. 设置用户邮箱 (要注意user.email是key，key空格后的是需要修改的value值)
```java
  git config --global user.email "email@example.com"
```
3. 最后查看git设置列表信息
```java
  git config --list 
```
4. 查看用户名
```java
  git config user.name 
```
