### Java中的乱码解决一般思路

```
乱码的解决思路和进制之间的转换相似
 
比如八进制的20想转换成二进制
一般思路：八进制的20先转换成十进制，然后十进制再转换成二进制

****解决乱码的问题，前提是要先知道生成这个乱码的数据是什么字符集编码出来的才行****

如果说有一个字符集 ISO8859-1编码的结果是 ？？？？ 想转换成UTF-8字符集的数据
一般思路：先将？？？？这个值按原先的ISO8859-1字符集编码获取到字节码数组，
然后再将这个字节码数组进行想要的字符集UTF-8编码输出即可，这样就解决了乱码的问题

JAVA代码实现：

String beforeResult = "？？？？"；
byte[] bytes = beforeResult.getBytes("ISO8859-1");
String afterResult = new String(bytes, "UTF-8");
```
